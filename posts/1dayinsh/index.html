起因的話毛象的朋友也有看到，就是週四晚上十點左右我看到📕上長期關注的一個獵頭博主「布布糕」突然發了拜龍華寺很靈驗的視頻，不知該說是我突然被龍華寺召喚還是他平時視頻太有趣的原因（就算是玄學內容也對我很有說服力），於是盤算著週五有個調休，要不去看看？ 之前看過那麼多杭州靈隱寺的視頻，我一次也沒真正想飛過去，誰知道這次就大腿一拍去了呢。

當然臨時起意+完全沒做功課導致我定的機票和酒店都在浦東，這成了我寫作「短途旅行」讀作「突擊拉練」的行程開端。

前方多圖（真的超級多圖）預警。

龍華寺半日遊
#

去龍華寺的路程本身就花了不少時間。酒店在祝橋於是先坐酒店班車去機場，由機場搭2號線轉去龍華寺，11點出發大概下午1點多才到。在門前可以取香，有免費的隨緣香也有20元3支請的香，其實內容都是一樣的，講究一個心誠則靈。

虔誠拜拜
#

跟著人流繞塔三週後進去山門就可以自由拜拜了。和你蓉部分寺廟很不一樣的是佛像旁有大量的鮮花供俸，工作人員也會一直注意花束狀況進行更換。整體環境都非常清幽舒服，不知道是不是因為我週五去所以人不算多。

繞塔時我有學著別人在心裡默唸身份證號+姓名，倒是忘了有沒有像「扔掉煩惱」一樣把香擲進香爐。真正跪在大雄寶殿前的時候幾欲流淚，可能有為行動力突然變強的自己的感動，也有成功到達佛祖菩薩面前的喜悅T T想了下我祈願的內容其實也很簡單，如果諸位菩薩（尤其是普賢菩薩…）能夠聽到我的心聲的話，便也不算白來了這一趟。

拜拜完去大會堂給二老請了手串，沒想到手串又是什麼三合又是八大講究還挺多的。唯一沒請的是太歲手串，畢竟太歲我印象中算是，道教那邊的…

名聲在外的素麵也吃了！可能是工作日去的原因，沒遇到📕上說的排隊，看師傅直接痛快一大勺澆頭淋入碗中我就在悄悄吞口水（x 味道也是真的好，清淡鮮甜，香菇豆泡我都很喜歡。
隨緣素菜味道就普普。不過據說麵和菜都是可以續的，沒什麼好挑剔的了。

好多貓，真的好多貓
#

比起僧人，貓更像是龍華寺的主人，悠然自得地四處轉悠，來人逗弄也頗淡定，抬眼看看就繼續沈浸在自在小天地了。也有不少好心人會帶著貓糧貓條投餵，貓也不客氣只管低頭吃吃吃。不知不覺間樹上長出一隻貓，拜墊上長出一隻貓，香爐旁也有…柱子上也有…這裏難道是貓貓天堂？

接下來請沈浸式欣賞鏡頭中的一些貓貓：

⬆️這一隻小貓的圖我個人很喜歡，頗有禪意，旁邊牌匾上書「有禪有淨土，猶如帶角虎」，下半句是「現世為人師，來生作佛祖」，小貓又伏臥於蓮花柱上安睡，哎就很自在安逸...
⬆️ ⬇️ 這兩隻其實是同一隻貓，不知道怎麼拍出來感覺這麼不一樣
⬇️ 這隻就有點臭臉，一直在看草叢中的另外一隻狸花，不過那隻天色暗了我沒拍到...
放一些零零散散的其他的🐱：

當然作為狗黨在寺內也看到了很精神的兩小隻，動作實在太整齊劃一了忍不住拼了個圖：

其實還有一隻土狗（？）是躺在殿門口曬太陽睡覺的，結果有兩位女士摸頭就算了一直很大力在rua佢，估計是煩了，佢爬起來嘆了口氣頭也不回往殿內找了個陰影處趴下繼續睡了。我還沒來得及拍到佢…
人壞！

怎麼還有鳥啊
#

對除了四處可見的貓貓還有鳥和烏龜。拜拜完別人都踱出去了只有我抱著相機在龍華塔下看鵝鴨打架（鵝鴨殺？）

磨磨蹭蹭拍完差不多也快到下午五點，和術士約好吃飯，於是坐地鐵過去面基✌️

上海，你真的太大了
#

上次和術士見面已經是去年夏天的事了，只記得在你蓉熱得東倒西歪嗦冰粉閒聊很開心。這次在上海相見，我居然在這個天氣整成了熱傷風（當時還不覺得只以為是嗓子因為霧霾發炎，後來反應過來了：純粹是跑來跑去穿得還很厚給熱出來的）…
作為一個在你蓉對著導航都能迷路的路癡，先到地方十分鐘後還沒找到店招牌的我已經開始坐立難安，微信篤篤狂敲：媽呀是這裡嗎
術士：哈哈哈哈等我馬上來！
趁著等人走去旁邊路上逛了會兒，7仔似乎有賣我沒在你蓉見過的東西。欸不是這瓶怎麼沒見過，這瓶也沒有…
喝完看了看旁邊似乎有位女士還在抱著電腦敲表格，於是又想到了白天逛完龍華寺蹲門口打工作電話的事情，哎。

吃點喝點
#

第二次見面雖然聊的都是些工作瑣事有的沒的，但充滿了工驢忙裡偷閒的快樂。術士帶俺吃的這家雲南菜也挺合我胃口，米線很清淡剛好適合，木薑子炒牛肉超級香，咬破了木薑子之後香氣在口中瀰漫開來又很清爽，好味好味。

另外仍要感嘆術士果然很有大姐姐的氣場———（悄悄）

\短暫做會兒二次元/
#

晚飯完畢我說可以跟著術士去做二次元嗎，術士非常熟練地帶我去了新世界和百聯ZX…在新世界我還在感嘆你滬怎麼一個商場能到8樓9樓實在很難想像（一般不都6層樓就完了嗎！）潮玩星球正在搞人狼村之謎的主題餐廳，還有HUNTER✖️HUNTER的主題咖啡廳…不過我這兩部都沒怎麼關注（。
走到百聯ZX的時候拍了個照發給群內兩位廣東的大親友，兩位大親友異口同聲地：大城市就是好啊。

百聯ZX給我印象很好，人流量不小的情況下新風開得夠足，當時嚴重鼻塞的我整體感受還算比較舒適，佈局和分區也算合理，當然也是因為整體內容豐富有空間來規劃而不是單純地賣谷，這點吧你蓉有些地方比如天府紅請反省一下———哎感受到了一些差距。
Aniplex店內除了招牌fgo部分還有經典ip的小走廊我很喜歡！以及這次托術士的福我還拿了購物送的fgo紙袋…後來發現是上海限定，夯啊。

前往外灘！
#

跟著迷迷糊糊走到了外灘…看照片定位途中應該是走到了南京路步行街？其實那裡給我強烈的春熙路即視感，該說是全國哪裏的商業街都差不多嗎…甚至連街道佈局都很類似。但城市風貌整體還是差距太大了，抬頭拍照看到高樓林立我都忍不住自嘲一句鄉毋寧( ´･ω) （x
路過了chiikawa shop，給親友拍了兩張，可愛還是可愛的。

上一次專門來外灘好像已經最起碼十幾年前（欸我居然也會這麼說）關於那個時間只記得好熱好難受為什麼要大熱天在你滬走，黃浦江黃黃的，看東方明珠也不過爾爾。這次晚上來看感受又不一樣，很平和又安心，奇異的感受。
有幾分可惜是我太拖沓導致走到外灘已經熄燈了沒有拍到勝景。但在薄霧中和術士一起壓馬路的感受挺奇妙的，一邊走感嘆上海原來如此之大一邊又覺得在這個時間點能和朋友一起在外灘聊天也是講求因緣際會的事情，至少此刻我很幸運。
留一些遺憾和心願下次再實現吧。

逛完已經十點半2號線停運了…和術士一起狂奔坐了10號線末班車之後我打車去了虹橋機場然後坐機場大巴去浦東機場再坐酒店的末班接機車回酒店，到酒店就凌晨1:30了。痛苦慘叫。
上海還是太大太大了。

滴水湖2小時遊
#

我去的這兩日天氣都不怎麼好，第一天大霧霾第二天也沒大太陽，原本去滴水湖是看很多人去那邊拍浮光躍金（不是）夕陽之類的結果到了之後有種大受騙的感覺！飛機落地浦東機場之前也有在空中看到這一個大大的圓，當時還在疑惑這到底是哪裡，沒想到第二天搜「浦東機場周邊景點」就有了答案，原來是超大型人造湖啊…考慮到我下午就要趕飛機只有3-4個小時的自由活動時間，掃個共享單車開始環湖！

滴水湖旁櫻花盛開迎風招展的樣子實在是太美了…淡粉色的花瓣如雲似霞分外美麗，忍不住在樹下看了好久好久。

騎著騎著天就陰下來了風也逐漸轉涼，湖水顏色也變得昏暗。

湖邊隨處可見的是隨波漂流的水鳥。我先開始沒認出來是什麼，說是鴨子呢嘴又是尖的，說是雞呢又能在水中漂遊…後來去社交網路搜索了一下，發現是「白骨頂雞」⬇️叫聲很怪但很悅耳，wer wer wer的哈哈哈，感覺鳥類知識多了一點點。而且似乎是來上海越冬的鳥類，又感覺自己很幸運。

還有比較有趣的是騎到濕地附近的時候看到有位女士在端著手機對濕地拍什麼，出於人類的好奇我仔細一看，是一隻白貓伺機而動盯上了一隻，白鷺…不過幾番嘗試之後都沒有成功。
我拍得樂呵呵的，貓似乎都察覺到我在笑佢，回頭掃了我一眼XD

滴水湖在地圖上顯示位於臨港新區，附近的高樓大廈很多看著也很氣派，但總覺得沒什麼配套也沒什麼人，儘管大屏幕一直在播什麼Young City for the Youth的宣傳語，還是感覺人氣不旺比較冷清，大概定位是郊區中的郊區、給年輕牛馬工作的地方吧...

拍完騎完發現路程大概9km，環湖走走停停騎了1h左右，靠湖的橋上區域很多是不可以上自行車的我也就沒有騎上去，橋上也許會有不一樣的風景呢。

說實話騎完感覺凍成傻子了，怕趕不上飛機一路狂奔去龍陽路地鐵站扒了兩口淮揚菜什麼的（…）就是另一個故事了。

下次再見
#

總而言之是相當匆忙的一次行程，（甚至返程也是慌慌張張怕錯過登機，慌到在上海借了個充電寶忘記還，直到前幾天才發現。）但旅行可能就是有一些遺憾才好，下一次希望能做好計劃更悠然地和朋友們相見，看更多的風景（以及吃更多好吃的，做更多遊客要做的事（x

下次再見吧上海。不知道下次去是什麼時候…（希望，希望是去龍華寺還願的時候。）
就拜託普賢菩薩了🙏🥹
