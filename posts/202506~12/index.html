青箬笠，绿蓑衣，斜風細雨不須歸

你蓉六七月的雨都是一陣一陣的下，某日想起來：又到寫博客的時候啦，腦內就突然想到這句詩。當然決定用這個標題的時候我也未曾預料到，淅淅瀝瀝的溫柔小雨到7月初會變成暴雨傾盆（甚至發展到你蓉居民可以上街劃船的地步）。
哎我就是很喜歡在奇怪的地方引用…
當然我更沒想到的是寫完這篇的時候一看日曆已然年底了，這四五個月都在幹什麼啊。
那麼，以下是大型頭重腳輕的流水帳。另外俺在思考，過兩天要不要寫一個，年度的，洗髮總結呢…。

六月初：俺也終於去過三星堆了
#

雖然是你蜀人（嗯…古蜀和現在的「蜀」似乎也沒什麼太大關係了）但是作為此地一大景點，一直沒有去過總覺得好像不太正常。拜託那可是三星堆欸。
看的時候只會感嘆好大的裝置、好細緻的設計、好精湛的技藝…
要說最大的收穫，可能是之前完全沒在意過的青銅器的分類的具體展現。在現場觀展從鼎、鬲再到尊、罍、卣等終於有了非常直觀的感受，花紋可能不仔細看都差不多，實際拆解描摹下來就能看到還是非常複雜大有不同，花紋非常精美形象。
請看點俺手機拍的圖。

運氣很好的是沒做任何功課直接衝去三星堆，趕上四號館正在辦布達拉宮特展。大部分是清代展品，也有部分元明代製品，金飾、織物等都很美。
隨便放幾張圖。

上圖這個明代的唐卡歷經歲月風霜在現代還能如此鮮豔沒有暈色，啊能看到這樣的展品真是好幸運啊…

說到打卡，在博物館昏暗環境拍照其實挺難拍的，於是我和男友選擇：

（解鎖一些荒謬的情侶打卡照）

相比之下成都博物館真是災難中的，災難。

七八月：疲憊中尋回生活秩序
#

情緒崩盤
#

可能是工作burnout或者激素帶來的影響，自我回顧一番發現下來七八月的狀態非常不好，很多時候能感受到精力被誰從身體中抽離出來，我躺在床上喘氣都會非常疲憊，真的好想一睡不醒。在這種狀況下看影視劇也好玩遊戲也好只會把我拽入更深的黑洞之中，身體本能讓我選擇不去做這些。

對於工作本身心態倒是已經調整得相對平和，但也只能算是相對平和，依然是爛泥中掙扎的心情。如果反覆做一件事情沒有正面回饋只是碰壁，我還要一直去嘗試嗎？不管是嘗試喜歡人也好還是嘗試喜歡工作也好這都不算是非常好的體驗吧？偏偏一直都在做這種事，像撞南牆一樣。如果反覆失望的話我肯定也會進入「不想努力」的狀態的。

啊，往右邊一看同事大哥又在WoW搓木樁了…

八月下旬遇到了非常不好的事情，連續三天都處於非常恍惚、失落與無措的狀態，好在我的大腦有不錯的防禦機制，會自動幫我把非常痛苦的回憶打包放在角落。但人體機能確實是複雜至極，在很多被莫名trigger的瞬間，坐在工位我都需要努力忍住眼淚才能顯得正常。後來問題解決的時候我又在想以前看的書上有說：情緒來的時候讓它自然地流動，能讓情緒更快地過去。我也確實是這麼嘗試的，接受情緒也是本人修行的一部分，似乎真的能讓自己好受一些。

魂3，我終於通關了！
#

7月底把從頭到尾（幾乎每一分鐘）都自己修煉跑圖的號給打通了本篇，8月初磨過了DLC1雪原的芙莉德修女，打得我直喊反胃…八月中轉去打環印城，只覺得難但並不算噁心。黑龍米狄爾一直搓不過去很讓人沮喪，但是每天磨個1小時2小時的，終於還是過了，蓋爾大開大合也都算是可以躲過去的招數，所以推完DLC後男友問我心情如何，我也只是淡淡地：嗯終於打完了…沒有非常的喜悅或是激動，大概是因為這一週目的我（以及作為玩家的我）已經變強了不少吧。

總之終於通關了，可喜可賀可喜可賀👏

和好友的會面
#

七月下旬和八月底分別見了過來出差的大學的朋友（同時也是學姐）和小學同班男性好友，儘管自認彼此已經是非常親密的關係，在談話的時候也難免身體疲憊眼神游移。

我可能真的不是精力充沛的人，高強度的1-2h社交對精力不錯的人來說可能算不上什麼，對我來說已是莫大的消耗，這個我幾年前在自己微博上也提到過，也是很早就自我覺察到的事情。和好友的會面比較像是我從自己的鴕鳥洞中出來的一個契機，總要有這麼一次兩次會面讓我去感受到他人生活的溫度，去觀察和理解他人的行為模式，算是我這種內向型人（？）感知世界的途徑之一。

學姐比起大學的時候安靜了很多，和同事開啟的話題也多是關於家庭、伴侶等，有很多抱怨的內容如水淌過。一邊吃火鍋我一邊想：儘管大家都走上了不同的道路總會有不同的結局，但如果婚姻和生育會讓人變成這樣的話，那為什麼學姐和她的同事，或者說話題外的、我已經離婚的幾位朋友，在當時還要奮不顧身投入這樣的一個迷局呢？

看展，看展
#

看了兩個成都博物館的特展（嚴格來說，是2個特展+1個常駐展）

其一是 文明的史诗：来自地中海的古典雕塑艺术 。
特展價格多少忘了大概是49還是59一個人，價格高低暫且不論，買了特展票就不用再在公眾號預約搶入場名額這點是挺好的。展品以古典雕塑為主，可以完全沈浸進去，動線也合適。不過對我這種古羅馬/希臘歷史不甚了解的人來說，成博的名字混寫讓人會非常混淆？（比如「阿瑞斯（瑪爾斯）這種？）可能希臘神祇名給我印象更深刻一些的原因是小時候一定會看希臘神話但對羅馬那一段看得，不多…
在現場請老外扮神有點尷尬，真的。
另外就是展品的角度可以多樣一點就好了，我為了看清楚展品（不論是石棺還是雙耳瓶）都湊得好近好近好近最後拿相機在看（。
依然是手機拍圖，請欣賞。我最喜歡的還是雅典娜的雕像（…

石棺看起來非常生動，可惜距離太遠太遠了，我貼好近在看啊！
怎麼展陳設計這個石棺是凹進去的啊（哭笑不得）

其二是 大备盛德：青铜时代的曾国与鄂国 。

青銅器前後反反覆覆已看了不少（包括同時間段成都博物館正在做先秦時期的成都-現代成都的展覽，青銅禮器既是歷史上時間相差不算特別長的產物，那形制就大差不差），可從觀感來說這個特展從名稱到展品給人的感受又不太一樣，獸面紋和神面紋的結合感覺兼有神秘與威嚴之感，青銅的藍色鏽跡在燈光照射下格外迷人，據說是第一次來成博展出，看到真的挺幸福的。
從姬周起源到曾鄂兩國歷史，整體動線清晰，電子屏還可以點看各個大姓來源，適合我這種人來掃盲。但成博，你的燈真的太黑了，我看清了展品就看不清字，看清字就看不清展品……

以及整個展覽C位同時也是海報圖上的神面紋卣，忍不住想放一張近景，實在太美太美了，包括那個藍色的鏽印。

至於為什麼說是三個展呢，因為後面又看了成博常駐展，先秦時期的成都+漢代陶俑都有看到（那個很有名的説唱俑哈哈哈哈），再之後是近世篇，1840年鴉片戰爭至1949保路運動，努力餐場景也有完整復原，再對比看看人民公園那邊的現實場景，啊，努力餐…
不過我沒怎麼拍圖，太黑啦！如果是成都人我覺得這部分都是值得一看的。

看「失憶投捕」也是被友安利的，看的時候我因為不懂棒球規則挺懵的，但動畫製作真的很不錯，故事流暢（為什麼一直強調流暢，失去力氣的時候我對片子基礎要求就是流暢）在七八月這種提不起勁的日子裡我居然輕鬆看完了，很熱血又有酸澀的青春醍醐味（什麼…）請各位都去看「失憶投捕」感受一下青春的力量…老實說我很喜歡那種天才對自己的才能毫無意識、傷害到了沒有這樣才能的親友的設定（。

九至十月：疲於奔命，好生無聊
#

工作幾乎是九和十月全部內容，好在還是看了幾部電影的，書影音記錄不算完全空白。

捕風捉影
九月工作之餘最開心的莫過於月底去看了捕風捉影（還好沒下線！）沒想到可以看到成龍拍了一大堆爛片之後和梁家輝精彩演繹了這麼一齣大戲。劇本就是純粹爽文，節奏踩得不錯，如我在毛象所說看兩位打架像是在虐待老人，但打戲很俐落好看就足夠。
比較大兩個缺點：
1.澳門區域的戲全程標準國語（交雜英語）我印象中應該是廣東話+英語+葡語為主啊？聽標準國語真的很怪；
2.少點中文rap吧好怪啊！
（另外我對JUN好感度蠻高的，結果出鏡還是一直戴口罩啊…）

快乐到死
韓國電影看得不多，給我印象大多是：色調陰鬱、氛圍惆悵像有雨後泥土潮濕的味道，這部也不例外。影片本身就簡單講了個背德故事，沒什麼深刻寓意，但全度妍女士太美麗了無法從她身上把眼睛挪開。 （百想影後全度妍女士和我推好幾次合作我都沒看，嘿） 看完在想當面對家庭和激情需要兩相抉擇的時候，男與女的選擇會一樣嗎？啊於是又陷入一些無端的惆悵。

十月好像就是在一直做活動執行，但空閒時還是抓了兩三部電影來看。

蜂蜜檸檬蘇打
求求日本人，放下對濕髮瀏海造型的執念吧…
另外看這個片子意外種草了一款片中的室內鞋，和友聊了好久說真好看很想要，不過放在購物車裏很快就忘了。

美食總動員
儘管動畫呈現很可愛但還是會和生活中的野鼠聯繫起來，無法和現實生活中的老鼠形象分離啊啊啊啊，看的時候像在左右腦互搏。感覺自己錯過了最佳觀影時機，導致現在看就像是無趣的大人在找回一點點可憐的童心otz

長安的荔枝
沒看過原作來看電影，本來不喜歡大鵬的長相覺得不夠美型苦哈哈的（…）但看完發現這長相還挺貼合角色特質，觀影體驗整體也流暢舒服，尤其是到影片後半同為社畜的體驗也許更為深刻：實誠人做事也許做得認真精確，但不一定被領導賞識；當歷盡艱辛甚至突破個人底線終於完成上級交代的任務的時候，站在上級面前還有力氣去問一聲值不值得嗎？

十一至十二月：遺願清單包含留下人生照片嗎
#

這麼寫標題的原因是：十一月我去拍了好幾次照片。

為了更新簡歷照片臨時起意隨便找了個樣片還算符合自己審美的照相館拍了headshot，首頁的朋友可能有瞟到（？），結果光想著好看忘記還要凸顯「專業」，選錯服裝選成了露肩的衣服。雖然成片效果真的如我所期望和要求的那樣「像個ABC」「稍微聰明一些」，但選衣的失誤讓我不得不把自己的帶臉照片上傳到ai讓他給我，換成，西服，再貼到簡歷裡。
畢竟不管中文還是英文簡歷我都不覺得招聘方能接受露肩裝（笑）。

整套照片修完我很滿意，下午為了不浪費這個層層打底的妝容還拉著男友去Photowith一通狂按快門（拍完發現妝都花得差不多了），收穫照片七十餘張，怪我不夠創意擺pose。

另外就是前段時間和二老拍了合影，照相館當面展示粗修照片，明顯是ai修圖磨皮磨得臉上肌肉動態都沒了把我氣得爆炸，母上卻很執拗地說她挺喜歡這版就要這版，甚至要自己付錢不讓我付說「妳給了我會覺得好像欠了什麼，我不高興」。
那時我突然意識到母上不是不愛美，拍照的時候她也很開心（雖然折騰起來很累）只是她好像也很不好意思在日常找一個契機提說想拍寫真想留下一些特別的影像，母上過去就很少直接表達喜好，最近兩年才表達得多了些。或許她執意付錢也只是想擁有那套當下很符合自己審美的照片罷了。

自問不算是個自戀的人，以前更不愛給自己拍照，多是給別人拍，以至於翻看相冊根本沒辦法完全拼湊起成長的軌跡。最近兩年逐漸有了「多留下一些關於自己（和家人也好，戀人也好）的影像」的想法，希望日後回顧的時候不至於遺憾。不是經常說人的大腦是一塊硬盤嗎，記憶和學習的事物都會反覆讀取也會被新的知識和記憶覆蓋，如果沒有別的介質來存儲想來我特別怨念的事情應該是前幾年得空的時候沒能去西安拍漢服照片（其實本地照相館不是沒有好的妝造和服裝以及攝影，但有的事情不去特定的地方做總感覺少了些樂趣）後來又過勞肥加上沒空休假就更不可能實現了，這個念想一直在心裡沒能放下。

同理還有JK制服照，現在都進入中年人隊列了好像拍JK制服照也成了不太合時宜的事情，不知道日後有無實現的機會。

並未忘記觀影
#

眼科醫院的錢真好賺.jpg
#

以下段落不構成醫學建議，僅為個人記錄。

我有輕度乾眼症我一直都知道（算是普遍病症）也一直沒太在意，眼藥水（玻璃酸鈉/氯化鈉）偶爾才買來點一下。某日晚上開始突然覺得眼睛有強烈異物感，對鏡檢查卻沒有任何異物（至少我看不到）。翻上下眼瞼只能看到大量紅血絲，自查不算沙眼也沒麥粒腫，猜測可能是結膜炎於是光速購入托百士的妥布黴素滴眼液，第二日早晨充血徵狀明顯好轉，但異物感依然存在，就開始計劃週末就醫，結果到了下午我就忍不住衝去醫院了。實在是磨眼睛，畏光和流淚症狀同時出現真的有點害怕。

給醫生說了情況和目前用藥之後檢查果然發現雙眼都有結石，因此有輕度的角膜劃傷（看了機器拍照，眼睛內白色點狀物就是結石，就不放圖了）。右眼的結石據說都已經冒出來，左眼結石像新筍一般只露了個尖，醫生擔心創面太大不建議取出。

我：請問今天能完事嗎？
醫生：能的。那你先去做乾眼治療，然後再去旁邊治療室挑結石吧。挑完之後你這個就連點三天即可。
我：（摸索摸索掏出眼藥水）目前我在用的是參天散克巴維生素B12滴眼液 、樂敦清C3複方氯化鈉滴眼液，這些ok嗎？另外我在用妥布黴素。現在海露開不出來了是吧。
醫生看了一眼我手裏三瓶眼藥水可能覺得很好笑，沈吟幾秒後：對開不出來了，你只有自己去買喔。然後你這個…我給你換個產品吧。
我：我其實有在猶豫要不要買小牛血，但我沒處方。
醫生：？你有小牛血？有的話就點那個修復角膜很好的啊。
…喔？

乾眼治療是熟悉的瞼板腺按摩，先蒸汽熏蒸放鬆雙眼，然後用夾子夾眼睛下緣處的瞼板腺，將堵塞的油脂擠出。醫師也說我油脂狀況良好並沒怎麼堵塞，只能說明我不是堵塞型瞼板腺功能障礙吧。負責去除結石的醫師也非常好心，右眼處理完畢（感覺有五六顆）後看我左眼下方的結石較為明顯，也順手幫我處理了。
說一下給開的單。

西藥费 273.53

  • 茲潤環孢素滴眼液(II) ：165
  • 思然聚乙二醇滴眼液 ：67.71
  • 速高捷小牛血去蛋白提取物眼用凝膠 ：40.82

檢查费 18
治療麻醉費 30
乾眼症治療（熏蒸+瞼板腺按摩） 134

這家醫院的乾眼症治療是自費項目………我不知道其他醫院是怎麼算，但單獨現金自付一百多大洋還是會肉痛的。環孢素這個藥吧也是不知道為什麼這麼貴…據說之前更貴現在進醫保還算降價了。

洗髮水空瓶
#

夏秋季消耗速度真是驚人。

Mild By Nature豐盈洗髮水
#

  • 兩性離子表活（椰油酰胺丙基羥磺基甜菜鹼）複配 月桂酰羥甲基乙磺酸鈉 (SLMI)
  • 香味：柑橘香
  • 價位：65-85RMB／354mL
  • 起泡度：★★★★
  • 洗後清爽感：★★★★★

這個也是用了很久的產品，一般會拿來作為交替洗中強力清潔的那一瓶。成分沒啥問題，清潔力好所以不能每日使用。

洗完反饋就是哇頭髮好乾好掛手，真的非常乾淨（洗護手法正確的話並不會怎麼掉髮，在正常水平內），也沒有明顯打結。有的去油洗髮水洗完又打結髮尾又乾澀，劉海位置還依然是塌下去的，但這款就不會，大概也是因此叫做「豐盈」洗髮水吧。個人建議是得搭配護髮素使用。

至於香味就是檸檬葉的感覺，我完全沒覺得是柑橘味道。

蜂花生薑健髮洗髮露
#

  • SLES表活（月桂醇聚醚硫酸酯鈉）複配 兩性離子表活（ CAPB 椰油酰胺丙基甜菜鹼）
  • 香味：啤酒花香
  • 價位：39.9RMB／750mL
  • 起泡度：★★★★
  • 洗後清爽感：★★★★★

沒想到在7月用完了這一瓶。

表活加得不少，S系表活為主，原料便宜大碗。洗的時候很乾爽吹的時候也很乾爽，問題在於：洗完之後會隱隱約約的癢…
幾年前用的時候懷疑是自己沒洗乾淨，後來有了洗髮的一些基礎知識之後就反應過來了，原來是S系表活作怪。這個成分敏感一些的用家洗完就是會頭皮癢+起疙瘩，如果在意這個表活成分那我建議就是：別買，別給自己找不痛快。

當然健康的頭皮交替洗用這個產品也是沒問題的，因為他去油效果真的很不錯，洗完也不算太乾燥。第二天如果濕度大的話還是會覺得略塌，細軟髮質的用家也要注意。
留香是個值得誇獎的點，清清爽爽的味道能持續個兩天左右。
